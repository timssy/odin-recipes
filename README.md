# odin-recipes

## Learnings in Intro of HTML  of The Odin Project
- Basics of HTML such as structure of the html, html tags (e.g. p tags, anchor tags, img tags, heading tags)
```html
<!DOCTYPE html>
<html>

    <!-- head is where the additional metadata or information of the webpage placed. --> 
    <head>
        <!-- title of the webpage. It can be seen in the tab of the web browser --> 
        <title>Title</title> 
    </head>

    <!-- body is where HTML documents are located. It is the one that is presented in the browser -->
    <body>
        <!-- HTML headings consist of h1 to h6. h1 tag is the most important heading and h6 tag is the least important heading -->
        <h1>Headings</h1>

        <!-- p tags are for paragraphs -->
        <p>Paragraphs</p>

        <!-- Ordered list - listed the items in numberical or alphabetical -->
        <ol>
            <li>List 1</li>
            <li>List 1</li>
        </ol>

        <!-- Unordered list - listed the items bullet marks -->
        <ul>
            <li>List 1</li>
            <li>List 1</li>
        </ul>

        <!-- img tags are for images. src attribute specifies the location of the image -->
        <img src="">

        <!-- anchor tags are for links. href attribute specifies the location of the link -->
        <a href="">
    </body>
</html>
````
